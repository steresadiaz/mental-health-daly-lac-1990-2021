# Social determinants of mental health burden in Latin America (1990–2021)

This repository contains the code used for the analysis of a longitudinal panel study examining the association between structural determinants and mental health burden (DALYs) in Latin America and the Caribbean.

## Data sources
- Global Burden of Disease (IHME)
- World Development Indicators (World Bank)

## Methods
- Panel data analysis (country-year)
- Fixed-effects models
- Driscoll–Kraay standard errors
- Non-parametric trend analysis (Mann–Kendall, Theil–Sen)

## Reproducibility
All data used in this study are publicly available. Code is provided to reproduce the main analyses.
